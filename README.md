### 📌 Featured Projects

**🚕 Real-Time Streaming Pipeline — Uber Ride Analytics on Azure**
End-to-end streaming pipeline using Event Hubs (Kafka) and a FastAPI microservice with Faker-based synthetic ride data. Implemented a 3-tier Medallion Lakehouse on Databricks — Bronze (Kafka ingest), Silver (streaming joins), Gold (CDC star schema, SCD Type 1 & 2) — using PySpark Structured Streaming with event-time watermarking and Delta Lake CDC.

**🏥 Real-Time Streaming Pipeline — Healthcare Revenue Cycle Management**
Azure Data Factory pipelines ingesting multi-source healthcare data into ADLS Gen2 (full + incremental loads), with CDC-based ETL in Databricks producing ACID-compliant Delta Lake Silver tables. Gold-layer star schema and Power BI dashboards for revenue cycle KPIs, secured with Azure Key Vault and audit logging.

**🌍 Master Thesis — Global Refined Petroleum Trade Forecasting**
Azure Data Factory ingestion pipeline for UN Comtrade petroleum datasets (100+ countries), with SARIMAX forecasting and network analysis modelling regional trade flows through 2028.
